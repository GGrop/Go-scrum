# Go Scrum

Go Scrum is a To-do Task demo, I learned much about libraries like skeleton, form with formik and validation with Yup.


 #### Click here 👉🏻 [Demo](https://ggrop.github.io/Goscrum/)

## Install

```bash
  npm install
```

## Tecnologies


```bash
  React.js
  React-router-dom
  React-loading-skeleton
  sweetalert2
  Redux
  Formik
  Yup
  GitHub-pages
  framer-motion
  mui material
```
